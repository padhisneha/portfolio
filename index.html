BikerX

BikerX is a bike servicing platform using Dart Flutter, Firebase, and Google Maps API. It offers comprehensive maintenance services, including scheduling appointments, tracking service history, and receiving reminders.

  • Technologies Used: Dart, Flutter, Firebase, Google Maps API
  • Features: Appointment scheduling, Service history tracking, Reminders
  • Role: Mobile App Developer

Pothole Detection System

The Pothole Detection System uses dashcam footage and CNN models to identify potholes on roads. It leverages numpy, matplotlib, Python, TensorFlow, Keras, and OpenCV to process and analyze the data.

  • Technologies Used: Python, TensorFlow, Keras, OpenCV
  • Features: Real-time pothole detection, Data visualization
  • Role: Data Scientist

HealthHub

HealthHub is a telemedicine kiosk designed to provide remote healthcare services. The system integrates video conferencing, electronic medical records, and diagnostic tools to facilitate virtual consultations.

  • Technologies Used: React, Node.js, MongoDB
  • Features: Video conferencing, EMR integration, Diagnostic tools
  • Role: Team Lead, Front-end Developer
